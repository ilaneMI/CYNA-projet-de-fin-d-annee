## 🎯 Objectif

Créer une application **moderne, mobile-first et sécurisée** permettant :

- L’achat de services SaaS  
- La gestion des abonnements  
- L’administration via un back-office  

---

## 🚀 Fonctionnalités principales

- Catalogue de services  
- Panier et paiement sécurisé  
- Gestion des abonnements  
- Espace administrateur  
- Chatbot & formulaire de contact
